## Installation
- Download the plugin on **Spigot** or **GitHub** and place it in your ***"plugins"*** folder.
- Restart your server, configure the plugin.

### Conflicts
* The following plugins cannot be used with KubLock
    - BlockProt
    - Chest Protect
